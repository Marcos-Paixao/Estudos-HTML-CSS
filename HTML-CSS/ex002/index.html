Parágrafos e quebras de linhas

Voce pode escrever um parágrafo em várias linhas no código fonte que o navegador irá exibir tudo em uma única linha.

Para criar um novo parágrafo, você deve usar a tag <p>.

Se voce precisar quebar o texto em algum ponto, use a tag <br>.
Ela cria uma quebra de linha.
